# technical-challenge-fullstack-engineer
Este desafio visa avaliar suas habilidades como Engenheiro(a) Full-Stack na construção de uma aplicação moderna, funcional e resiliente. O objetivo é desenvolver uma “Smart To-Do List”: uma lista de tarefas que integra Inteligência Artificial para decompor objetivos de alto nível (ex: “Planejar uma viagem”) em subtarefas acionáveis.
